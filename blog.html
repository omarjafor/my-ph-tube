Discuss the scope of var, let and const ?

Var: Variable decleared with var is a function scoped. Var are hoisted to the top of the function like global scope. Var can be reassigned.

Let: Let is a blocked scoped that means it is block in a decleared function and cannot be used in outside of the function. Variable decleard with let can be reassigned but can not be redecleared again.

Const: Const are also a blocked scoped and cannot be reassigned or can not be change the value

Tell us the use case of null and undefine?

Null:
1. To represent a value that is intentionally assigned but there is no value we use null.
2. To avoid unexpected errors can be use null.

Undefine:
1. To represent a missing value or not initialized values are undefine.
2. If any function does not return are undefine.
3. Function parameters that is not passed is also undefine.
4. An object property that is not exist in object are undefine.
5. Index number that doesn't exist in an array are also undefine.
